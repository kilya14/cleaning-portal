# Документация проекта «Мой Не Сам»

## ER-диаграмма

Откройте **`er-diagram.html`** в браузере — отобразится ER-диаграмма базы данных.

Чтобы сохранить как изображение:
- В Chrome: правая кнопка по диаграмме → «Сохранить изображение как…»
- Или используйте DevTools → скриншот области

## Скриншоты мобильной версии (390×844 px)

Откройте **`mobile-screenshots-instructions.html`** — там пошаговая инструкция по созданию скриншотов мобильной версии в Chrome/Firefox.

Страницы для съёмки:
1. Авторизация (index.php)
2. Регистрация (register.php)
3. Личный кабинет (dashboard.php)
4. Создание заявки (create-order.php)
5. Админ-панель (admin/index.php)
